Project Overview

An IoT-based smart classroom system designed to automate energy management and attendance monitoring.

This project focuses on developing an intelligent classroom automation system using STM32 microcontroller and IoT technology.

The system integrates RFID-based attendance monitoring, environmental sensing and automatic control of classroom appliances.

Sensors such as DHT11 and BH1750 are used to monitor temperature, humidity and lighting conditions while reducing unnecessary power consumption.
